🛠️ 技术实现

主要功能特性

  • 视频播放器：支持HTML5视频播放，带有封面图和播放控制
  • 一键开玩：直接启动游戏的快捷按钮
  • 双重描述：游戏描述 + 录屏描述的分离设计
  • 相关推荐：展示相关热门视频的网格布局
  • 社交功能：点赞、分享、观看次数统计
  • 响应式设计：适配不同屏幕尺寸

文件结构

  • src/pages/community/video.vue - 主视频页面组件
  • src/router.ts - 添加了视频页面路由
  • src/components/video/VideoItem.vue - 视频项组件 (已存在)
  • src/apis/video.ts - 视频API接口 (已存在)
  • src/mock/videoData.ts - 模拟数据 (已存在)

🔗 访问方式

视频页面的访问路径遵循RESTful设计：

/video/{owner}/{name}

例如：

/video/demo_user/my-first-game

路由函数

// 获取视频页面路由 export function getVideoPageRoute(owner: string, name: string) { return `/video/${encodeURIComponent(owner)}/${encodeURIComponent(name)}` }
